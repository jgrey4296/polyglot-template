# .csproj

To have a root project and exclude a tests folder:

```xml
<ItemGroup>
    <Compile Remove="tests/*"/>
</ItemGroup>
```

While Test subprojects incude a reference:
```xml
<ItemGroup>
    <ProjectReference Include="$(MSBuildThisFileDirectory)../{}.csproj" />
</ItemGroup>
```
