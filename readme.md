# Polyglot

A multi-language [Cookiecutter](https://cookiecutter.readthedocs.io/en/stable/README.html) template.

Covers:
- Python with [uv](https://github.com/astral-sh/uv)
- [Elixir](https://elixir-lang.org/) using [mix](https://hexdocs.pm/mix/1.18.4/Mix.html)
- Rust with [cargo](https://doc.rust-lang.org/cargo/)
  - Rust <-> Python with [maturin](https://www.maturin.rs/)
  - Rust <-> Emacs Lisp with [emac](https://docs.rs/emacs/latest/emacs/) crate.
- [dotnet](https://learn.microsoft.com/en-us/dotnet/)
- Lisp with [eask](https://emacs-eask.github.io/)
- [Kotlin](https://kotlinlang.org) with [Gradle](https://docs.gradle.org/current/userguide/userguide.html) and and [sdkman!](https://sdkman.io/)
- [Rocq](https:://rocq-prover.org) with ocaml
- Prolog in [swipl](https://www.swi-prolog.org/)
- Ansprolog in [clingo](https://potassco.org/)
- [Jacamo](https://github.com/jacamo-lang/jacamo) in [Kotlin](https://kotlinlang.org/docs/home.html) 
- [Soar](https://github.com/SoarGroup/Soar)
- [Godot](https://docs.godotengine.org/en/latest/index.html)
- [Octave](https://octave.org)
- [Open Policy Agent](https://www.openpolicyagent.org/)
- [Bash](https://www.gnu.org/software/bash/manual/bash.html)

With documentation generated by:
- [sphinx](https://www.sphinx-doc.org/en/master/contents.html) for python/general uses.
- [mdbook](https://rust-lang.github.io/mdBook/)
- [rustdoc](https://doc.rust-lang.org/rustdoc/what-is-rustdoc.html) for rust.
- [docfx](https://dotnet.github.io/docfx/) for dotnet.
- [dokka](https://kotlinlang.org/docs/dokka-cli.html#run-with-json-configuration) for kotlin.
- [doxygen](https://www.doxygen.nl/manual/)
- [texlive](https://www.tug.org/texlive/)
- Changlelogs with [towncrier](https://towncrier.readthedocs.io/en/stable/tutorial.html)
